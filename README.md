# Untis2RSS

A Node.js application that monitors your WebUntis timetable and provides updates via an RSS feed.

So far, the application supports:
- **Timetable Changes**: Tracks cancellations, substitutions, room changes, and irregular events.
- **Exams**: Notifications for upcoming exams.
- **Absences**: Alerts when an absence is recorded or excused.
- **Messages**: "Message of the Day" announcements from the school.
